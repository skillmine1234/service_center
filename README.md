service_center
==============

Service Center App

The following are the environment variables required to be set:

1. DEVISE_AUTHENTICATE_WITH_LDAP
2. CONFIG_URL_AML_SEARCH
3. FROM_EMAIL_ID 
4. SMTP_ADDRESS 
5. SMTP_USERNAME 
6. SMTP_PASSWORD 
7. CONFIG_FILE_UPLOAD_PATH
8. CONFIG_APPROVED_FILE_UPLOAD_PATH
9. CONFIG_LOGO_PATH
